# Buttons

Use Material Kit's React custom button styles for actions in forms, dialogs, and more with support for multiple states, and more.

You will find all of our restyled buttons from material-ui in `src/_material-toolkit-components/CustomButtons`.

## Styles
You will find the styles for this component in
`_assets/jss/material-kit-react/components/buttonStyle.js`
and
`_assets/jss/material-kit-react/components/iconButtonStyle.js.`

## Examples
Material Kit React has changed the predefined button styles from Material UI, each serving its own semantic purpose, with a few extras thrown in for more control.

```javascript
import badgeStyle from "assets/jss/material-kit-react/components/badgeStyle.js"
```

## Properties

### Controls

<Story story={buttonStories.Props} />

```javascript
RegularButton.propTypes = {
  classes: PropTypes.object.isRequired,
  backgroundColor: PropTypes.string, //must match hex or rgb color
  color: PropTypes.oneOf([
    "primary",
    "info",
    "success",
    "warning",
    "danger",
    "rose",
    "white",
    "facebook",
    "twitter",
    "google",
    "github",
    "transparent"
  ]),
  size: PropTypes.oneOf(["sm", "lg"]),
  simple: PropTypes.bool,
  round: PropTypes.bool,
  fullWidth: PropTypes.bool,
  disabled: PropTypes.bool,
  block: PropTypes.bool,
  link: PropTypes.bool,
  justIcon: PropTypes.bool
};
```
